# 🍔 Delicia Express - Delivery Autônomo

Sistema completo de delivery que roda **100% no GitHub Pages**, sem servidor.

## Pilares do Sistema

1. **`produtos.json`** - O Cérebro (cardápio completo)
2. **`admin.html`** - Painel de Controle Profissional
3. **`index.html`** - Experiência do Cliente (iFood style)

## Funcionalidades

### Painel Admin
- Personalização completa de cores
- Gestão de horários por dia da semana
- CRUD de categorias com reordenação
- Gestão de produtos (pausar, editar, excluir)
- Configuração de WhatsApp, taxa de entrega e nome da loja

### Loja para Cliente
- Design mobile-first premium
- Status em tempo real (aberto/fechado)
- Favoritos com ❤️
- Carrinho dinâmico com subtotal em tempo real
- Finalização via WhatsApp com mensagem formatada
- Busca e filtro por categoria

## Como usar

1. Faça upload de todos os arquivos para o GitHub
2. Vá em **Settings → Pages**
3. Escolha **Deploy from branch** → `main` → `/ (root)`
4. Aguarde 1 minuto

Acesse:
- Loja: `https://seuusuario.github.io/nomerepo/`
- Admin: `https://seuusuario.github.io/nomerepo/admin.html`

**PIN de acesso ao Admin: Não possui (acesso direto)**
